# AI Usage Report

## Tools Used and Use Cases

### ChatGPT

ChatGPT was used during this assignment for:

- planning how to extend the Assignment 2 portfolio into Assignment 3
- debugging the broken local file structure and mismatched file contents
- suggesting ways to combine API integration, state management, and complex UI logic
- improving documentation wording and organizing the project write-up

### GitHub Copilot

GitHub Copilot can be used as a code-completion assistant while writing HTML, CSS, and JavaScript. In this assignment context, it is most useful for:

- repetitive markup patterns
- styling boilerplate
- quick JavaScript scaffolding

## Benefits and Challenges

### Benefits

- AI helped generate implementation ideas faster.
- It made debugging easier by suggesting likely causes and structured fixes.
- It helped organize the documentation so the final report is clearer and more complete.
- It reduced time spent on repetitive code and formatting tasks.

### Challenges

- AI suggestions still needed review because not every suggestion matched the exact assignment rubric.
- Some suggestions were too generic and had to be adapted to this specific portfolio.
- It was important to verify logic carefully instead of trusting generated code immediately.

## Learning Outcomes

Using AI in this assignment improved:

- understanding of front-end state management with `localStorage`
- confidence in combining multiple JavaScript features in one page
- ability to break a large assignment into smaller implementation steps
- awareness of how to review and refine AI-generated suggestions instead of accepting them directly

## Responsible Use and Modifications

AI assistance was used responsibly by:

- reviewing all generated suggestions before adding them to the project
- modifying code to fit the portfolio design and assignment requirements
- checking that the logic, text, and file organization were correct
- rejecting generic or inaccurate outputs and replacing them with more suitable code
- documenting the use of AI transparently in this report

## Summary

AI was used as a support tool for planning, debugging, drafting, and refinement. The final implementation was still reviewed, edited, and organized to ensure correctness, originality, and academic integrity.
